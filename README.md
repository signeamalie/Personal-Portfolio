# SAJ Personal Portfolio

Et personligt portfolio udviklet som et professionelt visitkort og showcase for mine projekter som multimediedesigner.
Sitet er kodet fra bunden med fokus på struktur, genbrugelighed og brugeroplevelse - bygget i vanilla JavaScript (ES6 modules) og SCSS.

## Live version:
https://personalportfolio.signejensen.dk


## Formål
Portfolioen fungerer som et digitalt visitkort, der samler mine projekter i et konsistent, tilgængeligt og visuelt layout.
Målet er at give praktikvirksomheder og arbejdsgivere et klart overblik over mine kompetencer inden for:

Frontend-udvikling (HTML, SCSS, JS, OOP)

UX/UI & OOUX

Dataindsamling og SEO

Adobe Creative Cloud


## Kør lokalt

1.Klon repoet

2.Åbn mappen i VS Code

3.Start Live Server på index.html
(kræver VS Code Live Server extension)

4.SCSS kompileres automatisk til assets/styles/style.css


## Tech stack

JavaScript (ES6 modules) - OOP-struktur med klasser (Project, Skill, Slider)

SCSS -> CSS – modulopbygget struktur

Responsivt layout

Modulstruktur

/src
  /data
  /models
  /pages
  /ui-components
/assets
